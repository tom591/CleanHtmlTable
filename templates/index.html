CleanHtmlTable

An online converter that transforms styled HTML code into clean, plain HTML

CleanHtmlTable is a web application designed to clean HTML code by removing unwanted styles and unnecessary elements.

Will be removed: class, id, name, style, width, height, align, valign, nowrap, border, cellspacing, cellpadding
Will remain: table, tr, td, th, thead, tbody, a, br, p, h1, h2, h3, h4
